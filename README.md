# PhotoSnap — Galería de fotos responsive y minimal

PhotoSnap es una galería de fotos simple, bonita y responsiva que incluye un lightbox con navegación por teclado. Ideal como proyecto pequeño para practicar HTML/CSS/JS y para mostrar en tu portafolio.

Características
- Grid responsivo de imágenes
- Lightbox modal con navegación (← →) y cierre con Escape o clic fuera
- Transiciones suaves y diseño minimal
- Sin dependencias, funciona abriendo index.html o con GitHub Pages

Tecnologías
- HTML5, CSS3 (variables y grid/flexbox)
- JavaScript vanilla

Cómo usar (local)
1. Clona o crea un repo y copia estos archivos en la raíz.
2. Abre index.html en tu navegador.
3. Opcional: sustituye las URLs de ejemplo por tus imágenes.

Despliegue rápido
- Empuja el repo a GitHub y activa GitHub Pages desde la rama main (Settings → Pages).

Issues iniciales sugeridos
1. Inicializar proyecto: añadir index.html, styles.css, script.js y README. (good-first-issue)
2. Implementar lightbox modal y navegación por teclado.
3. Añadir imágenes de muestra y probar responsive en móviles.

Contribuir
- Crea un fork, abre un branch con tu cambio y haz un PR.
- Etiqueta cambios en CSS, JS o assets en la descripción del PR.
